# Library Management System in Java

A console-based Library Management System built using Java and Object-Oriented Programming concepts.

## Features

- Add new members
- Add books to library
- Borrow books
- Return books
- Delete accounts
- Authentication using passwords
- Display all books and members

## Concepts Used

- Object-Oriented Programming
- HashMap
- ArrayList
- File Handling
- Collections Framework

## Technologies

- Java
- Collections Framework

## Future Improvements

- Database integration
- GUI interface
